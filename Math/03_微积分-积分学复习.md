# 微积分-积分学复习材料

## 一、不定积分概念和基本积分公式

### 知识点讲解
**不定积分的定义**：若函数F(x)的导数等于f(x)，即F'(x) = f(x)，则称F(x)是f(x)的一个原函数。f(x)的所有原函数称为f(x)的不定积分，记作：
$$\int f(x)dx = F(x) + C$$
其中C为任意常数，称为积分常数。

**基本积分公式**：
- $\int x^n dx = \frac{x^{n+1}}{n+1} + C$ $(n \neq -1)$
- $\int \frac{1}{x} dx = \ln|x| + C$
- $\int e^x dx = e^x + C$
- $\int a^x dx = \frac{a^x}{\ln a} + C$
- $\int \sin x dx = -\cos x + C$
- $\int \cos x dx = \sin x + C$

### 选择题
**题目1**：下列哪个函数的不定积分是$\ln|x| + C$？
A. $\frac{1}{x}$  B. $\frac{1}{x^2}$  C. $\frac{1}{x+1}$  D. $\frac{1}{x-1}$

**答案：A**

**题目2**：计算$\int x^3 dx$的结果是：
A. $\frac{x^4}{4} + C$  B. $3x^2 + C$  C. $\frac{x^3}{3} + C$  D. $x^4 + C$

**答案：A**

### 解答题
**题目**：计算不定积分$\int (2x^3 - \frac{3}{x} + e^x) dx$

**解答过程：**
$$\int (2x^3 - \frac{3}{x} + e^x) dx = 2\int x^3 dx - 3\int \frac{1}{x} dx + \int e^x dx$$
$$= 2 \cdot \frac{x^4}{4} - 3 \ln|x| + e^x + C$$
$$= \frac{x^4}{2} - 3 \ln|x| + e^x + C$$

**答案**：$\frac{x^4}{2} - 3 \ln|x| + e^x + C$

## 二、换元积分法

### 知识点讲解
**换元积分法**是计算积分的重要方法，通过适当的变量代换，将复杂的积分转化为简单的积分。

**第一换元法（凑微分法）**：
若$\int f(u)du = F(u) + C$，且$u = g(x)$，则：
$$\int f(g(x))g'(x)dx = \int f(u)du = F(u) + C = F(g(x)) + C$$

**第二换元法**：
设$x = g(t)$，其中$g(t)$单调可导且$g'(t) \neq 0$，则：
$$\int f(x)dx = \int f(g(t))g'(t)dt$$

### 通俗讲解
**换元积分法的直观理解**：
想象你在解一个复杂的迷宫，直接走很困难。但如果你能找到一个"传送门"（新变量），让你先传送到一个简单的地方，在那里找到出口（计算积分），然后再通过传送门回到原来的地方，这就是换元法的思想。

比如$\int 2x(x^2+1)^3 dx$，直接展开会很复杂。但如果我们设$u = x^2 + 1$，那么$du = 2x dx$，原积分就变成了$\int u^3 du$，这就很简单的$\frac{u^4}{4} + C$，最后再把$u$换回$x^2 + 1$。

### 选择题
**题目1**：计算$\int 2x \sqrt{x^2+1} dx$，下列哪个替换最合适？
A. $u = \sqrt{x^2+1}$  B. $u = x^2+1$  C. $u = x^2$  D. $u = x$

**答案：B**

**题目2**：$\int \frac{1}{x+1} dx$通过换元法计算，设$u = x+1$，结果为：
A. $\ln|u| + C$  B. $\frac{1}{u} + C$  C. $u + C$  D. $\frac{u^2}{2} + C$

**答案：A**

### 解答题
**题目**：计算不定积分$\int \frac{x}{\sqrt{x-1}} dx$

**解答过程：**
设$t = \sqrt{x-1}$，则$t^2 = x-1$，$x = t^2 + 1$，$dx = 2t dt$

$$\int \frac{x}{\sqrt{x-1}} dx = \int \frac{t^2 + 1}{t} \cdot 2t dt = \int (t^2 + 1) \cdot 2 dt$$
$$= 2\int (t^2 + 1) dt = 2\left(\frac{t^3}{3} + t\right) + C$$
$$= 2\left(\frac{(x-1)^{3/2}}{3} + \sqrt{x-1}\right) + C$$
$$= \frac{2}{3}(x-1)^{3/2} + 2\sqrt{x-1} + C$$

**答案**：$\frac{2}{3}(x-1)^{3/2} + 2\sqrt{x-1} + C$

## 三、分部积分法

### 知识点讲解
**分部积分法**基于乘积的求导法则：
$$(uv)' = u'v + uv'$$
两边积分得：
$$\int (uv)'dx = \int u'v dx + \int uv'dx$$
即：
$$uv = \int v du + \int u dv$$
整理得：
$$\int u dv = uv - \int v du$$

### 通俗讲解
**分部积分法的直观理解**：
分部积分法就像"拆快递"的过程。你有一个复杂的"包裹"（被积函数），它由两部分组成：$u$和$dv$。有时候直接拆$dv$很困难，但如果你能找到$u$的"拆解方法"（求导），让$du$变得简单，同时$v$容易求得，那么就可以通过$uv - \int v du$来解决问题。

一个记忆技巧是"反对幂指三"顺序：
- **反**对数函数：$\ln x, \ln(x+1)$等
- **对**三角函数：$\sin x, \cos x$等  
- **幂**幂函数：$x^n$等
- **指**指数函数：$e^x, a^x$等

选择顺序靠前的作为$u$，因为它们的导数会变得更简单。

### 选择题
**题目1**：计算$\int x \sin x dx$，应设：
A. $u = x, dv = \sin x dx$  B. $u = \sin x, dv = x dx$  C. $u = x \sin x, dv = dx$  D. $u = 1, dv = x \sin x dx$

**答案：A**

**题目2**：$\int x e^x dx$使用分部积分法，设$u = x, dv = e^x dx$，则：
A. $du = dx, v = e^x$  B. $du = 1 dx, v = e^x$  C. $du = x dx, v = e^x$  D. $du = e^x dx, v = x$

**答案：A**

### 解答题
**题目**：计算定积分$\int_0^1 x \ln(x+1) dx$

**解答过程：**
使用分部积分法，设$u = \ln(x+1)$，$dv = x dx$
则$du = \frac{1}{x+1} dx$，$v = \frac{x^2}{2}$

$$\int_0^1 x \ln(x+1) dx = \left[\frac{x^2}{2} \ln(x+1)\right]_0^1 - \int_0^1 \frac{x^2}{2} \cdot \frac{1}{x+1} dx$$
$$= \frac{1}{2} \ln 2 - \frac{1}{2} \int_0^1 \frac{x^2}{x+1} dx$$

计算$\int_0^1 \frac{x^2}{x+1} dx$：
$$\frac{x^2}{x+1} = x - 1 + \frac{1}{x+1}$$
$$\int_0^1 \frac{x^2}{x+1} dx = \int_0^1 \left(x - 1 + \frac{1}{x+1}\right) dx = \left[\frac{x^2}{2} - x + \ln(x+1)\right]_0^1$$
$$= \left(\frac{1}{2} - 1 + \ln 2\right) - (0 - 0 + \ln 1) = -\frac{1}{2} + \ln 2$$

因此：
$$\int_0^1 x \ln(x+1) dx = \frac{1}{2} \ln 2 - \frac{1}{2}\left(-\frac{1}{2} + \ln 2\right) = \frac{1}{4}$$

**答案**：$\frac{1}{4}$

## 四、定积分定义和性质

### 知识点讲解
**定积分的定义**：
$$\int_a^b f(x)dx = \lim_{\lambda \to 0} \sum_{i=1}^n f(\xi_i)\Delta x_i$$
其中$\lambda = \max\{\Delta x_1, \Delta x_2, \ldots, \Delta x_n\}$是所有子区间长度的最大值。

**定积分的基本性质**：
1. **线性性**：$\int_a^b [af(x) + bg(x)]dx = a\int_a^b f(x)dx + b\int_a^b g(x)dx$
2. **区间可加性**：$\int_a^c f(x)dx + \int_c^b f(x)dx = \int_a^b f(x)dx$
3. **比较定理**：若$f(x) \leq g(x)$，则$\int_a^b f(x)dx \leq \int_a^b g(x)dx$
4. **估值定理**：$m(b-a) \leq \int_a^b f(x)dx \leq M(b-a)$，其中$m$和$M$分别是$f(x)$在$[a,b]$上的最小值和最大值

### 选择题
**题目1**：根据定积分的性质，$\int_0^1 (2x + 1)dx$的值等于：
A. $2\int_0^1 x dx + 1$  B. $2\int_0^1 x dx + \int_0^1 1 dx$  C. $\int_0^1 2x dx + \int_0^1 1 dx$  D. 以上都对

**答案：D**

**题目2**：若$f(x)$在$[0,1]$上满足$1 \leq f(x) \leq 2$，则$\int_0^1 f(x)dx$的取值范围是：
A. $[1, 2]$  B. $[0, 2]$  C. $[1, 1]$  D. $[0, 1]$

**答案：A**

### 解答题
**题目**：利用定积分定义计算$\int_0^1 x^2 dx$

**解答过程：**
将$[0,1]$区间$n$等分，每个子区间长度为$\Delta x = \frac{1}{n}$，取右端点$\xi_i = \frac{i}{n}$，则：
$$\int_0^1 x^2 dx = \lim_{n \to \infty} \sum_{i=1}^n \left(\frac{i}{n}\right)^2 \cdot \frac{1}{n}$$
$$= \lim_{n \to \infty} \frac{1}{n^3} \sum_{i=1}^n i^2$$
$$= \lim_{n \to \infty} \frac{1}{n^3} \cdot \frac{n(n+1)(2n+1)}{6}$$
$$= \lim_{n \to \infty} \frac{(n+1)(2n+1)}{6n^2} = \lim_{n \to \infty} \frac{(1+\frac{1}{n})(2+\frac{1}{n})}{6} = \frac{2}{6} = \frac{1}{3}$$

**答案**：$\frac{1}{3}$

## 五、牛顿-莱布尼茨公式

### 知识点讲解
**牛顿-莱布尼茨公式**（微积分基本定理）：
若$f(x)$在$[a,b]$上连续，$F(x)$是$f(x)$的任意一个原函数，则：
$$\int_a^b f(x)dx = F(b) - F(a)$$

这个公式建立了定积分与不定积分之间的联系，是计算定积分的重要工具。

**常用记号**：$F(b) - F(a) = [F(x)]_a^b = F(x)|_a^b$

### 选择题
**题目1**：$\int_0^1 e^x dx$的值是：
A. $e-1$  B. $e$  C. $e+1$  D. $1-e$

**答案：A**

**题目2**：$\int_1^2 \frac{1}{x} dx$的计算结果是：
A. $\ln 2$  B. $\ln 1$  C. $1$  D. $2$

**答案：A**

### 解答题
**题目**：计算定积分$\int_0^{\pi/2} \sin x dx$

**解答过程：**
$\sin x$的原函数是$-\cos x$，根据牛顿-莱布尼茨公式：
$$\int_0^{\pi/2} \sin x dx = [-\cos x]_0^{\pi/2}$$
$$= -\cos(\pi/2) - (-\cos 0) = -0 - (-1) = 1$$

**答案**：$1$

## 六、广义积分

### 知识点讲解
**广义积分**（反常积分）包括无穷积分和瑕积分。

**无穷积分**：
- $\int_a^{+\infty} f(x)dx = \lim_{b \to +\infty} \int_a^b f(x)dx$
- $\int_{-\infty}^b f(x)dx = \lim_{a \to -\infty} \int_a^b f(x)dx$
- $\int_{-\infty}^{+\infty} f(x)dx = \lim_{a \to -\infty} \lim_{b \to +\infty} \int_a^b f(x)dx$

**瑕积分**：
若$f(x)$在$x = a$或$x = b$处无界，则：
- $\int_a^b f(x)dx = \lim_{t \to a^+} \int_t^b f(x)dx$（$x = a$为瑕点）
- $\int_a^b f(x)dx = \lim_{t \to b^-} \int_a^t f(x)dx$（$x = b$为瑕点）

### 选择题
**题目1**：广义积分$\int_1^{+\infty} \frac{1}{x^2} dx$的值是：
A. $1$  B. $\frac{1}{2}$  C. $\frac{1}{3}$  D. 发散

**答案：A**

**题目2**：$\int_0^1 \frac{1}{\sqrt{x}} dx$是：
A. $0$  B. $1$  C. $2$  D. 发散

**答案：C**

### 解答题
**题目**：计算广义积分$\int_0^{+\infty} x e^{-x} dx$

**解答过程：**
使用分部积分法，设$u = x$，$dv = e^{-x} dx$
则$du = dx$，$v = -e^{-x}$

$$\int_0^{+\infty} x e^{-x} dx = \lim_{b \to +\infty} \left[-x e^{-x}\right]_0^b + \lim_{b \to +\infty} \int_0^b e^{-x} dx$$
$$= \lim_{b \to +\infty} \left(-b e^{-b} + 0\right) + \lim_{b \to +\infty} \left[-e^{-x}\right]_0^b$$
$$= 0 + \lim_{b \to +\infty} (-e^{-b} + 1) = 1$$

**答案**：$1$

## 七、定积分应用

### 知识点讲解
**平面图形面积**：
- 由曲线$y = f(x)$，$x = a$，$x = b$围成的图形面积：$S = \int_a^b |f(x)|dx$
- 由两条曲线$y = f(x)$，$y = g(x)$，$x = a$，$x = b$围成的图形面积：$S = \int_a^b |f(x) - g(x)|dx$

**旋转体体积**：
- 绕$x$轴旋转：$V = \pi \int_a^b [f(x)]^2 dx$
- 绕$y$轴旋转：$V = 2\pi \int_a^b x f(x) dx$

**平面曲线弧长**：
- $y = f(x)$在$[a,b]$上的弧长：$L = \int_a^b \sqrt{1 + [f'(x)]^2} dx$

### 选择题
**题目1**：曲线$y = x^2$与直线$y = x$所围成的图形面积为：
A. $\frac{1}{6}$  B. $\frac{1}{3}$  C. $\frac{1}{2}$  D. $1$

**答案：A**

**题目2**：曲线$y = \sqrt{x}$从$x = 0$到$x = 4$的弧长是：
A. $\frac{4}{3}$  B. $\frac{8}{3}$  C. $\frac{16}{3}$  D. $8$

**答案：B**

### 解答题
**题目**：求曲线$y = e^x$与$y = e^{-x}$及$x = 1$所围成的图形面积

**解答过程：**
首先确定交点：
- $y = e^x$与$y = e^{-x}$交于$(0, 1)$
- $y = e^x$与$x = 1$交于$(1, e)$
- $y = e^{-x}$与$x = 1$交于$(1, e^{-1})$

在$[0, 1]$区间内，$e^x > e^{-x}$，所以图形面积为：
$$S = \int_0^1 (e^x - e^{-x}) dx$$
$$= \left[e^x + e^{-x}\right]_0^1 = (e + e^{-1}) - (1 + 1) = e + \frac{1}{e} - 2$$

**答案**：$e + \frac{1}{e} - 2$

## 总结

积分学是微积分的重要组成部分，掌握好基本概念和计算方法至关重要。记住：
1. **不定积分是微分的逆运算**，验证答案的最好方法就是求导
2. **选择合适的积分方法**：观察被积函数形式，灵活运用换元法和分部积分法
3. **定积分的计算**要特别注意上下限的对应关系
4. **广义积分必须计算极限过程**，判断收敛性
5. **应用题要仔细确定积分上下限**，画出图形帮助理解

练习是提高积分计算能力的关键，多做多练，熟能生巧！
