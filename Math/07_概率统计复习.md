# 概率统计复习材料

## 1. 概率的基本概念和性质

### 知识点讲解
概率是描述随机事件发生可能性大小的数值，取值范围在[0,1]之间。概率具有三个基本性质：

**非负性**：对于任意事件A，都有P(A) ≥ 0
**规范性**：必然事件的概率为1，即P(Ω) = 1
**可列可加性**：对于互不相容的事件序列{A₁, A₂, ...}，有P(∪ᵢ₌₁^∞ Aᵢ) = ∑ᵢ₌₁^∞ P(Aᵢ)

### 选择题

**题目1**：设A、B为两个随机事件，若P(A) = 0.6，P(B) = 0.3，且A、B互不相容，则P(A∪B)等于：
A. 0.9
B. 0.8
C. 0.7
D. 0.6

**答案：A**

**详细分析**：
- **解题思路**：互不相容事件的概率可以直接相加
- **关键步骤**：P(A∪B) = P(A) + P(B) = 0.6 + 0.3 = 0.9
- **易错点**：容易误用条件概率公式或包含-排斥原理，但互不相容时直接相加即可

**题目2**：抛掷一枚均匀硬币3次，则恰好出现2次正面的概率为：
A. 1/8
B. 2/8
C. 3/8
D. 4/8

**答案：C**

**详细分析**：
- **解题思路**：使用二项分布概率公式
- **关键步骤**：P(X=2) = C₃² × (1/2)² × (1/2)¹ = 3 × 1/4 × 1/2 = 3/8
- **易错点**：容易忘记组合数C₃² = 3，或将指数位置搞错

### 解答题

**题目**：某班级有男生30人，女生20人，现随机抽取3名学生，求：
(1) 恰好抽到2名男生的概率
(2) 抽到的3名学生中至少有1名女生的概率

**解答**：
(1) 设X为抽取的男生人数，则X服从超几何分布
P(X=2) = C₃₀² × C₂₀¹ / C₅₀³ = (435 × 20) / 19600 ≈ 0.444

(2) 方法一：P(至少1名女生) = 1 - P(无女生) = 1 - P(3名男生)
P(3名男生) = C₃₀³ / C₅₀³ = 4060 / 19600 ≈ 0.207
所以P(至少1名女生) ≈ 1 - 0.207 = 0.793

方法二：P(至少1名女生) = P(1女2男) + P(2女1男) + P(3女)
= C₃₀¹×C₂₀²/C₅₀³ + C₃₀¹×C₂₀²/C₅₀³ + C₂₀³/C₅₀³ ≈ 0.793

**解题思路**：超几何分布适用于"不放回抽样"问题
**关键步骤**：正确识别分布类型，熟练使用组合数计算
**易错点**：容易将超几何分布误认为二项分布

## 2. 随机变量和概率分布

### 知识点讲解
随机变量是定义在样本空间上的实值函数，将随机试验的每个结果映射到一个实数。根据取值方式的不同，随机变量分为：

**离散随机变量**：只能取有限个或可列无限个值
- 概率分布：P(X = xᵢ) = pᵢ，其中∑pᵢ = 1

**连续随机变量**：取值充满某个区间
- 概率密度函数：f(x) ≥ 0，∫₋∞^∞ f(x)dx = 1
- P(a ≤ X ≤ b) = ∫ₐᵇ f(x)dx

### 选择题

**题目1**：设随机变量X的概率密度函数为f(x) = {2x, 0 ≤ x ≤ 1; 0, 其他}，则P(X > 0.5)等于：
A. 0.25
B. 0.5
C. 0.75
D. 1

**答案：C**

**详细分析**：
- **解题思路**：连续随机变量的概率通过积分计算
- **关键步骤**：P(X > 0.5) = ∫₀.₅¹ 2x dx = [x²]₀.₅¹ = 1 - 0.25 = 0.75
- **易错点**：忘记积分，或将积分上下限搞反

**题目2**：离散随机变量X的分布列为：
| X | 0 | 1 | 2 |
|---|---|---|---|
| P | 0.3 | 0.5 | 0.2 |
则P(X ≥ 1)等于：
A. 0.3
B. 0.5
C. 0.7
D. 0.8

**答案：C**

**详细分析**：
- **解题思路**：离散随机变量的概率直接相加
- **关键步骤**：P(X ≥ 1) = P(X=1) + P(X=2) = 0.5 + 0.2 = 0.7
- **易错点**：容易遗漏某个取值，或将不等号方向搞错

### 解答题

**题目**：设随机变量X的概率密度函数为f(x) = {k(4-x), 0 ≤ x ≤ 4; 0, 其他}，求：
(1) 常数k的值
(2) P(1 ≤ X ≤ 3)
(3) X的分布函数F(x)

**解答**：
(1) 根据概率密度函数的规范性：∫₀⁴ k(4-x)dx = 1
∫₀⁴ (4-x)dx = [4x - x²/2]₀⁴ = 16 - 8 = 8
所以k × 8 = 1，得k = 1/8

(2) P(1 ≤ X ≤ 3) = ∫₁³ (1/8)(4-x)dx = (1/8)[4x - x²/2]₁³
= (1/8)[(12 - 4.5) - (4 - 0.5)] = (1/8)[7.5 - 3.5] = (1/8) × 4 = 0.5

(3) 分布函数F(x) = ∫₋∞ˣ f(t)dt
当x < 0时，F(x) = 0
当0 ≤ x ≤ 4时，F(x) = ∫₀ˣ (1/8)(4-t)dt = (1/8)[4t - t²/2]₀ˣ = (1/8)(4x - x²/2)
当x > 4时，F(x) = 1

**解题思路**：先利用规范性确定参数，再计算概率和分布函数
**关键步骤**：熟练掌握积分运算和分布函数的定义
**易错点**：积分计算错误，分布函数定义域判断错误

## 3. 常见概率分布

### 知识点讲解

**二项分布B(n, p)**：描述n次独立重复试验中成功次数的分布
- 概率质量函数：P(X = k) = Cₙᵏ pᵏ(1-p)ⁿ⁻ᵏ
- 期望：E(X) = np，方差：Var(X) = np(1-p)

**泊松分布P(λ)**：描述稀有事件发生次数的分布
- 概率质量函数：P(X = k) = λᵏe⁻λ/k!
- 期望：E(X) = λ，方差：Var(X) = λ

**正态分布N(μ, σ²)**：最重要的连续分布，中心极限定理的基础
- 概率密度函数：f(x) = (1/√(2π)σ)e^(-(x-μ)²/(2σ²))
- 期望：E(X) = μ，方差：Var(X) = σ²

### 通俗讲解

**正态分布**可以想象成钟形曲线，大多数数据都集中在平均值附近，越往两端概率越小。比如人的身高、考试成绩等都近似服从正态分布。标准正态分布N(0,1)是特殊的正态分布，任何正态分布都可以通过标准化Z = (X-μ)/σ转换为标准正态分布。

**泊松分布**常用来描述"稀有事件"，比如一天内接到的电话次数、一本书中的印刷错误数等。特点是平均值等于方差。

**二项分布**描述的是"n次独立重复试验中成功次数"，比如抛硬币10次正面朝上的次数、产品质量检测中的不合格品数量等。

### 选择题

**题目1**：若X ~ B(20, 0.4)，则E(X)等于：
A. 8
B. 10
C. 12
D. 16

**答案：A**

**详细分析**：
- **解题思路**：直接使用二项分布期望公式
- **关键步骤**：E(X) = np = 20 × 0.4 = 8
- **易错点**：容易将p和(1-p)搞混，或忘记乘以n

**题目2**：设X ~ P(3)，则Var(X)等于：
A. 3
B. 6
C. 9
D. 12

**答案：A**

**详细分析**：
- **解题思路**：泊松分布的方差等于参数λ
- **关键步骤**：Var(X) = λ = 3
- **易错点**：容易将方差误认为λ²或其他值

### 解答题

**题目**：某电话交换机每分钟接到的呼叫次数X ~ P(4)，求：
(1) 某一分钟内恰好接到3次呼叫的概率
(2) 某一分钟内接到呼叫次数不超过5次的概率
(3) 连续观察10分钟，至少有8分钟接到呼叫次数不超过5次的概率

**解答**：
(1) P(X = 3) = 4³e⁻⁴/3! = 64 × e⁻⁴/6 ≈ 0.195

(2) P(X ≤ 5) = P(X=0) + P(X=1) + ... + P(X=5)
= e⁻⁴(1 + 4 + 8 + 32/3 + 32/3 + 32/15) ≈ 0.785

(3) 设Y为10分钟内接到呼叫次数不超过5次的分钟数
则Y ~ B(10, 0.785)
P(Y ≥ 8) = P(Y=8) + P(Y=9) + P(Y=10)
= C₁₀⁸(0.785)⁸(0.215)² + C₁₀⁹(0.785)⁹(0.215) + (0.785)¹⁰
≈ 0.123 + 0.257 + 0.082 = 0.462

**解题思路**：熟练使用泊松分布概率公式，理解二项分布在连续观察中的应用
**关键步骤**：正确计算泊松概率，识别独立重复试验场景
**易错点**：泊松分布参数λ的物理意义理解错误，组合数计算错误

## 4. 数学期望、方差、协方差

### 知识点讲解

**数学期望**：
- 离散型：E(X) = ∑xᵢpᵢ
- 连续型：E(X) = ∫₋∞^∞ xf(x)dx

**方差**：Var(X) = E(X²) - [E(X)]² = E[(X-μ)²]

**协方差**：Cov(X,Y) = E(XY) - E(X)E(Y)
**相关系数**：ρ(X,Y) = Cov(X,Y)/[σₓσᵧ]

**重要性质**：
- E(aX + b) = aE(X) + b
- Var(aX + b) = a²Var(X)
- Var(X + Y) = Var(X) + Var(Y) + 2Cov(X,Y)
- 若X、Y独立，则Cov(X,Y) = 0，Var(X + Y) = Var(X) + Var(Y)

### 选择题

**题目1**：设随机变量X的分布列为P(X=1) = 0.3，P(X=2) = 0.7，则E(X²)等于：
A. 1.7
B. 2.3
C. 3.1
D. 4.3

**答案：C**

**详细分析**：
- **解题思路**：使用E(X²) = ∑xᵢ²pᵢ计算
- **关键步骤**：E(X²) = 1² × 0.3 + 2² × 0.7 = 0.3 + 2.8 = 3.1
- **易错点**：容易将E(X²)误认为[E(X)]²

**题目2**：若X、Y相互独立，E(X) = 2，E(Y) = 3，Var(X) = 1，Var(Y) = 4，则E(2X + 3Y)和Var(2X + 3Y)分别为：
A. 13, 22
B. 13, 40
C. 13, 46
D. 13, 52

**答案：B**

**详细分析**：
- **解题思路**：利用期望和方差的线性性质
- **关键步骤**：
  E(2X + 3Y) = 2E(X) + 3E(Y) = 2×2 + 3×3 = 13
  Var(2X + 3Y) = 4Var(X) + 9Var(Y) = 4×1 + 9×4 = 40
- **易错点**：独立时方差不能直接相加的系数搞错

### 解答题

**题目**：设二维随机变量(X,Y)的联合分布列为：
| Y\X | 1 | 2 | 3 |
|-----|---|---|---|
| 0   | 0.1 | 0.2 | 0.1 |
| 1   | 0.2 | 0.3 | 0.1 |

求：(1) E(X), E(Y), E(XY)
(2) Var(X), Var(Y)
(3) Cov(X,Y), ρ(X,Y)

**解答**：
(1) 先求边际分布：
P(X=1) = 0.1 + 0.2 = 0.3
P(X=2) = 0.2 + 0.3 = 0.5
P(X=3) = 0.1 + 0.1 = 0.2

P(Y=0) = 0.1 + 0.2 + 0.1 = 0.4
P(Y=1) = 0.2 + 0.3 + 0.1 = 0.6

E(X) = 1×0.3 + 2×0.5 + 3×0.2 = 1.9
E(Y) = 0×0.4 + 1×0.6 = 0.6
E(XY) = 1×0×0.1 + 2×0×0.2 + 3×0×0.1 + 1×1×0.2 + 2×1×0.3 + 3×1×0.1 = 0.2 + 0.6 + 0.3 = 1.1

(2) E(X²) = 1²×0.3 + 2²×0.5 + 3²×0.2 = 0.3 + 2.0 + 1.8 = 4.1
Var(X) = 4.1 - 1.9² = 4.1 - 3.61 = 0.49

E(Y²) = 0²×0.4 + 1²×0.6 = 0.6
Var(Y) = 0.6 - 0.6² = 0.6 - 0.36 = 0.24

(3) Cov(X,Y) = E(XY) - E(X)E(Y) = 1.1 - 1.9×0.6 = 1.1 - 1.14 = -0.04
ρ(X,Y) = Cov(X,Y)/√(Var(X)Var(Y)) = -0.04/√(0.49×0.24) ≈ -0.117

**解题思路**：先求边际分布，再计算各阶矩，最后求协方差和相关系数
**关键步骤**：熟练掌握边际分布的计算，理解协方差的定义
**易错点**：边际分布计算错误，相关系数的计算公式记错

## 5. 大数定律和中心极限定理

### 知识点讲解

**大数定律**：当试验次数足够多时，样本均值依概率收敛于总体期望
- 切比雪夫大数定律：样本均值依概率收敛于总体期望
- 伯努利大数定律：频率依概率收敛于概率

**中心极限定理**：大量独立同分布随机变量的和近似服从正态分布
- 林德伯格-列维定理：若X₁, X₂, ..., Xₙ独立同分布，E(Xᵢ) = μ，Var(Xᵢ) = σ²，则∑Xᵢ近似服从N(nμ, nσ²)
- 标准化形式：(∑Xᵢ - nμ)/(σ√n) 近似服从N(0,1)

### 通俗讲解

**大数定律**就像"水满则溢"的道理：虽然每次抛硬币的结果不确定，但抛的次数越多，正面出现的频率就越接近0.5。这告诉我们，偶然中包含着必然，重复试验可以揭示规律。

**中心极限定理**是统计学最重要的定理之一。可以想象成"木桶效应"：虽然每个木板的长度有高有低，但当木板数量足够多时，总高度会趋向于正态分布。这解释了为什么很多自然现象都近似服从正态分布。

简单来说：单个随机变量的分布可能很复杂，但多个随机变量的和却趋向于正态分布，这就像是"众多个体的合力"会趋向于稳定的结果。

### 选择题

**题目1**：设X₁, X₂, ..., Xₙ为独立同分布的随机变量序列，E(Xᵢ) = μ，Var(Xᵢ) = σ²，则根据中心极限定理，∑Xᵢ近似服从的分布为：
A. N(μ, σ²)
B. N(nμ, σ²)
C. N(nμ, nσ²)
D. N(μ, nσ²)

**答案：C**

**详细分析**：
- **解题思路**：直接应用中心极限定理
- **关键步骤**：和的期望为nμ，方差为nσ²
- **易错点**：容易将方差误认为σ²或其他形式

**题目2**：抛掷一枚均匀硬币1000次，根据大数定律，正面出现的频率：
A. 严格等于0.5
B. 以概率1收敛于0.5
C. 依概率收敛于0.5
D. 不确定

**答案：C**

**详细分析**：
- **解题思路**：理解大数定律的收敛方式
- **关键步骤**：大数定律保证的是依概率收敛，不是必然收敛
- **易错点**：容易将"依概率收敛"理解为"必然收敛"

### 解答题

**题目**：某工厂生产零件，已知每个零件的重量X ~ N(100, 4)（单位：克），现随机抽取100个零件，求：
(1) 这100个零件总重量的分布
(2) 总重量超过10150克的概率
(3) 若要求总重量以95%的概率不超过某个值，求该值

**解答**：
(1) 设Y = X₁ + X₂ + ... + X₁₀₀，其中Xᵢ ~ N(100, 4)且相互独立
根据中心极限定理，Y ~ N(100×100, 100×4) = N(10000, 400)

(2) P(Y > 10150) = P((Y-10000)/20 > (10150-10000)/20)
= P(Z > 7.5) ≈ 0（查表可知几乎为0）

(3) 设阈值为a，则P(Y ≤ a) = 0.95
标准化：(a - 10000)/20 = 1.645（标准正态分布95%分位数）
解得：a = 10000 + 1.645×20 = 10332.9克

**解题思路**：正确应用中心极限定理，熟练使用标准化和查表
**关键步骤**：识别独立同分布条件，正确计算和的均值和方差
**易错点**：中心极限定理的应用条件，标准化时忘记除以标准差

## 6. 参数估计和假设检验基础

### 知识点讲解

**点估计**：
- 矩估计：用样本矩估计相应的总体矩
- 极大似然估计：选择使样本出现概率最大的参数值

**区间估计**：给出参数估计的置信区间
- 置信水平：1 - α（如95%置信水平）
- 置信区间：(θ̂ₗ, θ̂ᵤ)

**假设检验**：
- 原假设H₀和备择假设H₁
- 检验统计量和临界值
- 第一类错误（α错误）：原假设为真时拒绝
- 第二类错误（β错误）：原假设为假时接受

### 通俗讲解

**参数估计**就像"盲人摸象"：我们只能看到大象的一部分（样本），但要推断整头大象（总体）的特征。通过合理的统计方法，我们可以从局部推断整体。

**点估计**给出的是一个具体数值，比如"这个班级的平均身高是170cm"；**区间估计**给出的是一个范围，比如"这个班级的平均身高在165-175cm之间，置信度为95%"。区间估计更能反映估计的不确定性。

**假设检验**像是"法庭审判"：我们先假设被告无罪（原假设），然后收集证据（样本数据），如果证据足够强，就拒绝原假设（判有罪）。但要记住，我们可能犯两类错误：错判无罪者为有罪，或错判有罪者为无罪。

### 选择题

**题目1**：设总体X ~ N(μ, σ²)，σ²已知，若要求μ的95%置信区间长度为L，则样本量n应满足：
A. n ≥ (2×1.96σ/L)²
B. n ≥ (1.96σ/L)²
C. n ≥ (2×1.96σ/L)
D. n ≥ 1.96σ/L

**答案：A**

**详细分析**：
- **解题思路**：利用置信区间长度公式
- **关键步骤**：置信区间长度为2×1.96×σ/√n = L，解得n ≥ (2×1.96σ/L)²
- **易错点**：容易忘记平方或搞错系数

**题目2**：在假设检验中，若显著性水平α = 0.05，则：
A. 第一类错误的概率为0.05
B. 第二类错误的概率为0.05
C. 置信水平为0.05

### 解答题

**题目**：某品牌电池的寿命X ~ N(μ, σ²)，随机抽取16节电池，测得寿命（小时）为：
280, 265, 275, 285, 290, 275, 270, 260, 295, 280, 275, 290, 270, 285, 275, 280

已知σ = 10小时，求：
(1) μ的点估计
(2) μ的95%置信区间
(3) 检验H₀: μ = 275 vs H₁: μ ≠ 275（α = 0.05）

**解答**：
(1) 样本均值x̄ = (280+265+...+280)/16 = 276.5625小时
点估计：μ̂ = x̄ = 276.56小时

(2) 95%置信区间：x̄ ± 1.96×σ/√n = 276.56 ± 1.96×10/4
= 276.56 ± 4.9 = (271.66, 281.46)小时

(3) 检验统计量：Z = (x̄ - μ₀)/(σ/√n) = (276.56 - 275)/(10/4) = 1.56/2.5 = 0.624
临界值：±1.96
因为|Z| = 0.624 < 1.96，所以接受H₀

**解题思路**：熟练掌握点估计、区间估计和假设检验的计算步骤
**关键步骤**：正确计算样本均值，熟练使用标准化公式
**易错点**：置信区间公式记错，假设检验的决策规则搞错
